# D-Guardian — Portfólio de Projeto

## Sobre o projeto
Este é um site estático criado como portfólio para a D-Guardian, uma empresa fictícia de segurança eletrônica com atuação no Nordeste do Brasil.

O site apresenta uma landing page completa com navegação fixa, seção sobre, serviços, galeria, mapa de atuação e depoimentos de clientes.

## Tecnologias usadas
- HTML5
- CSS3
- JavaScript puro
- Leaflet.js (mapas interativos)
- Git e GitHub
- GitHub Pages (para publicar o site na nuvem)

## Estrutura do projeto
- `index.html` – página principal com a estrutura do site
- `css/index.css` – estilos e layout para desktop e mobile
- `js/index.js` – interação do menu mobile, mapa Leaflet e scroll suave
- `img/` – pasta para imagens de apoio e fotos do projeto
- `.gitignore` – ignora arquivos locais e pastas como `.vscode/`

## Como o site foi estruturado
1. Navbar fixa com links de navegação interna e botão para WhatsApp
2. Banner hero com chamada principal e botões de ação
3. Seção "Sobre" com apresentação da empresa e benefícios
4. Seção de serviços com cards destacando soluções oferecidas
5. Galeria de imagens para mostrar trabalhos realizados
6. Mapa interativo usando Leaflet para destacar cidades atendidas
7. Depoimentos de clientes para reforçar confiança
8. CTA final para gerar contato via WhatsApp

## Deploy na nuvem
O projeto foi versionado com Git e enviado para o GitHub. A partir do repositório no GitHub, o site pode ser publicado como página estática usando GitHub Pages, tornando-o disponível na nuvem para visualização pública.

## Como usar
1. Abrir `index.html` localmente ou publicar no GitHub Pages
2. Editar as imagens em `img/` e substituir os placeholders nas seções de hero, sobre e galeria
3. Ajustar o número de telefone do WhatsApp se necessário
4. Atualizar as cidades no mapa pelo atributo `data-lat` / `data-lng` em `index.html`

## Contato
Para demonstrar o projeto na nuvem, basta compartilhar o link do GitHub ou da página hospedada no GitHub Pages.
